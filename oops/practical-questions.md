# OOPS Practical Questions

1. Write a Java program to demonstrate method overloading.
2. Write a Java program to demonstrate method overriding.
3. Write a Java program to implement encapsulation.
4. Write a Java program to demonstrate single inheritance and method overriding.
5. Write a Java program that uses a constructor, abstract class, and encapsulation in a single application.
6. Write a Java program to demonstrate compile-time polymorphism.
7. Write a Java program to demonstrate runtime polymorphism.
8. Write a Java program to demonstrate abstraction using an abstract class.
9. Write a Java program to demonstrate abstraction using an interface.
10. Write a Java program to demonstrate all access modifiers (public, private, protected, default).
11. Write a Java program to demonstrate constructor overloading.
12. Write a Java program to demonstrate the use of the final keyword with variable, method, and class.
13. Write a Java program to demonstrate the use of static variable and static method.
14. Write a Java program to demonstrate hierarchical inheritance.
15. Write a Java program to demonstrate multilevel inheritance.
16. Write a Java program to demonstrate method overriding with dynamic method dispatch.
17. Write a Java program to demonstrate polymorphism using parent class reference.
18. Write a Java program to demonstrate interface inheritance.
19. Write a Java program to demonstrate constructor.
